Documentation

Write documetation using Vim in md format. Use pandoc to generate html. A sample makefile to automate this and write it to site /doc/ folder is given below. The html format will be tracked by Git. All documentation that is not public goes here. Export html files to /doc/ under webroot. Public documentation goes under /pubdoc/. Examples of markdown formatting are given below.

Makefile

SOURCE_DOCS=$(wildcard *.md)
EXPORTED_DOCS=$(SOURCE_DOCS:.md=.html)
EXPORTED_DOCS_WWW=$(EXPORTED_DOCS:%=/var/www/html/cangjie.info/public_html/doc/%)

all: $(EXPORTED_DOCS_WWW)

PANDOC_OPTIONS=--standalone
PANDOC_HTML_OPTIONS=--to html5

/var/www/html/cangjie.info/public_html/doc/%.html : %.md
        pandoc $(PANDOC_OPTIONS) $(PANDOC_HTML_OPTIONS) -o $@ $<

.PHONY: all
